230. 二叉搜索树中第K小的元素

题目描述：
给定一个二叉搜索树的根节点rootrootroot ，和一个整数 kkk ，请你设计一个算法查找其中第 kkk 个最小元素（从 111 开始计数）。

二叉搜索树（BST：Binary Search Tree）：

  • 其左子树（left subtree）下的每个后代节点（descendant node）的值都小于节点nnn的值。
  • 其右子树（right subtree）下的每个后代节点的值都大于节点 nnn 的值。
    树的遍历 + 优先队列（堆）
    构建一个容量为 kkk 的大根堆
    根据大根堆的元素个数和当前节点与堆顶元素的关系来分情况讨论：
  • 大根堆元素不足 kkk 个：直接将当前节点值放入大根堆；
  • *大根堆元素为 kkk 个，根据堆顶元素和当前节点值的大小关系进一步分情况讨论：
  • 如果当前节点值元素大于堆顶元素，说明当前节点值不可能在第 kkk 小的范围内，直接丢弃；
  • 如果当前节点值元素小于堆顶元素，说明当前节点值可能在第 kkk 小的范围内，先 pollpollpoll 一个再 addaddadd 进去。

中序遍历
二叉搜索树的中序遍历是有序的，只需要中序遍历，然后返回第k个元素即可。

282. 给表达式添加运算符（难）

回溯算法
题目描述：
给定一个仅包含数字0−90-90−9的字符串numnumnum和一个目标值整数targettargettarget，在numnumnum的数字之间添加二元运算符（不是一元）+、−+、-+、− 或 ∗*∗，返回所有能够得到目标值的表达式。

回溯算法
设字符串numnumnum的长度为nnn,构建表达式时需要向nnn个字符的n−1n-1n−1个空间中插入+、−+、-+、− 或 ∗*∗，或者不添加符号。我们使用回溯来模拟。
定义递归方法dfs(intu,longcur,longpre,Strings)dfs(int u,long cur,long pre,String s)dfs(intu,longcur,longpre,Strings)其中：

  • uuu为当前枚举到第u个数字
  • curcurcur为当前表达式的计算结果
  • preprepre为最后一个连乘串的计算结果（乘法的优先级高，需要单独考虑）
  • sss为当前构建的字符串
    该递归函数分为两种情况：
  • 如果 u=nu=nu=n，说明表达式已经构造完成，若此时有 cur=targetcur=targetcur=target，则找到了一个可行解，我们将 sss 放入答案数组中，递归结束；
  • 如果 u<nu<nu<n，需要枚举当前表达式末尾需要添加的符号（+、−+、-+、− 或 ∗*∗），以及该符号要截取多少位，设当前符号后面的数字为valvalval
    • 若添加+++，则curcurcur增加valvalval，且valvalval为当前表达式的最后一个连乘串。
    • 若添加−-−，则curcurcur减少valvalval，且−val-val−val为当前表达式的最后一个连乘串。
    • 若添加∗*∗，由于乘法优先级高，所以需要和撤销preprepre的计算结果，并添加新的连乘结果pre∗valpre*valpre∗val,即cur−pre+pre∗valcur-pre+pre*valcur−pre+pre∗val.
      数字可以是单个000，但不能有前导零，需要单独判断，运算过程可能会有超过32位的整数，所以要用64位的整数存储运算结果。
